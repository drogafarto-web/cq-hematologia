## Escalation Matrix

### Data Issue Found

| Issue                                   | Severity | Action                           | Owner   | Slack               |
| --------------------------------------- | -------- | -------------------------------- | ------- | ------------------- |
| Missing fixture file                    | HIGH     | Stop phase execution, regenerate | QA Lead | #hc-quality-testing |
| Invalid JSON                            | CRITICAL | Fix + validate, re-run tests     | Backend | #hc-quality-testing |
| Data isolation breach (prod in staging) | CRITICAL | Quarantine data, audit, reset    | DevOps  | #incidents          |
| Test user creation fails                | HIGH     | Check Firebase Auth, recreate    | DevOps  | #hc-quality-testing |
| Fixture validation fails                | MEDIUM   | Investigate root cause, fix      | QA Lead | #hc-quality-testing |
| Auto-reset job fails                    | HIGH     | Manual reset, investigate logs   | DevOps  | #hc-quality-testing |

## Appendix: Script Templates

### Quick Reset Script (Copy-Paste)

```bash
#!/bin/bash
# reset-phase.sh

PHASE=${1:-4}
PROJECT="hmatologia2-staging"

echo "[$(date)] Resetting Phase $PHASE..."

# Delete data
firebase firestore:delete --recursive --project $PROJECT \
  --path "/labs/lab-00${PHASE}-staging"

# Load fixtures
node test/utils/load-fixtures.mjs --phase $PHASE --firebase-project $PROJECT

# Create users
node test/utils/create-test-users.mjs --phase $PHASE --firebase-project $PROJECT

# Validate
npm run test:validate-fixtures -- --phase $PHASE

echo "[$(date)] Reset complete for Phase $PHASE"
```

### Automated Nightly Reset (Cron)

```bash
# In scripts/nightly-reset.sh
0 2 * * * cd /path/to/hc-quality && bash scripts/nightly-reset.sh >> logs/nightly-reset.log 2>&1
```

### CI/CD Integration (GitHub Actions)

```yaml
name: Fixture Validation

on: [pull_request]

jobs:
  validate-fixtures:
    runs-on: ubuntu-latest
    steps:
      - uses: actions/checkout@v3
      - uses: actions/setup-node@v3
        with:
          node-version: 22
      - run: npm install
      - run: npm run test:validate-fixtures -- --all
```
