## Triagem de modelo (LLM operando este repo)

> Dica do CTO baseada em uso real (cota Opus queimou rápido em sessões mecânicas).

- **Default Sonnet 4.6** (cota abundante, cobre 90% do trabalho).
- **Opus 4.7** apenas para julgamento: arquitetura nova, debug sutil, review crítico, modelagem de domínio do zero, fechamento de sessão longa.
- **Subagentes sempre em Sonnet/Haiku** — pagar Opus pra subagente fazer grep é desperdício.
- **Plan → Opus, execução → Sonnet.** Pensa o plano em Opus, sai do plan mode, deixa Sonnet executar.

Equivalentes em outras ferramentas:
- **Cursor**: GPT-4o ou Claude Sonnet 4.6 padrão; Claude Opus 4.7 só pra arquitetura.
- **Codex CLI**: GPT-5 / Codex padrão; eleva quando trava.
