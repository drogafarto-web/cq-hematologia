# TDD Obrigatório

## Fluxo
1. Escreva o teste ANTES da implementação
2. Rode o teste — deve FALHAR (red)
3. Implemente o mínimo para passar
4. Refatore mantendo verde

## Stack de testes deste projeto
- Vitest para unit tests
- Firebase Emulator para integration tests
- Cobertura mínima: 80%
