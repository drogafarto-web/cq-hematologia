# Second Brain — Memória Persistente

## Objetivo
Manter contexto entre sessões sem depender do histórico de chat.

## Regras
- Antes de iniciar qualquer tarefa, leia ALWAYS: progress.md, decisions.md
- Ao finalizar uma tarefa, atualize decisions.md com decisões tomadas
- Ao encontrar um padrão recorrente, registre em patterns.md

## Arquivos de memória
- progress.md     → o que foi feito, o que está em andamento
- decisions.md    → decisões arquiteturais e seus motivos
- patterns.md     → padrões aprovados para este projeto
- context.md      → stack, módulos, convenções de nomenclatura
