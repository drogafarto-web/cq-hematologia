## Decision Gates & Approval Workflows

### Pre-Phase Kickoff Gate (Daily Check)

**Checklist before Phase N starts:**

- [ ] Phase N-1 complete & audited (if applicable)
- [ ] All dependencies locked (Phase N REQUIRES from roadmap)
- [ ] Executor assigned & briefed
- [ ] Compliance audit pre-signed (CTO + Auditor if Phases 4, 13–15)
- [ ] Risk register updated (RISK-NNN mitigation status)
- [ ] No P0/P1 blockers from previous phase

**Gate decision:** Phase lead + CTO → APPROVE | DEFER | CONDITIONAL

### Post-Phase Sign-Off Gate (Weekly on Fridays)

**Checklist after Phase N completion:**

- [ ] All acceptance criteria met (per REQUIREMENTS.md)
- [ ] Tests passing (≥95%)
- [ ] Code review approved
- [ ] Compliance audit passed (if regulatory)
- [ ] Artifacts published & linked in matrix above
- [ ] Risk register updated
- [ ] Downstream Phase N+1 unblocked

**Gate decision:** Phase lead + QA → SIGNED-OFF | REWORK | ESCALATE
