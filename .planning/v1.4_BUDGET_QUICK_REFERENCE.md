## 8. Cost Analysis Scripts

### Script A: Firebase Firestore Cost Breakdown (BigQuery)

```sql
-- Daily cost by Firestore operation
SELECT
  DATE(usage_start_time) AS date,
  sku.description,
  SUM(cost) AS daily_cost,
  SUM(usage.amount) AS usage_units
FROM `hc_quality_billing.gcp_billing_export_v1`
WHERE
  service.id = '00BE-609D-5E8D'  -- Firestore service ID
  AND DATE(usage_start_time) BETWEEN '2026-05-20' AND '2026-09-30'
  AND project.id = 'hmatologia2'
GROUP BY date, sku.description
ORDER BY date DESC, daily_cost DESC;
```

**Expected output:**
- Firestore Read Ops: $0.06 per 100k → ~$50/month baseline
- Firestore Write Ops: $0.18 per 100k → ~$15/month baseline
- Delete Ops: negligible
- Storage: $0.18/GB → ~$1.50/month

### Script B: Cloud Functions CPU Time Analysis (Logs)

```bash
# Query functions logs for latency
gcloud functions describe --region=southamerica-east1 \
  --format='table(name,runtime,status,sourceArchiveUrl)' \
  hmatologia2

# Per-function CPU time (requires logging setup)
gcloud logging read \
  "resource.type=cloud_function AND severity=INFO AND resource.labels.function_name:*" \
  --limit 1000 \
  --format=json \
  --project=hmatologia2 \
  | jq '.[] | {function: .labels.function_name, duration: .duration, cost_sec: (.duration * 0.000002556)}'
```

### Script C: Gemini Vision API Cost Check

```bash
# Check Gemini Vision usage logs
gcloud logging read \
  "resource.type=api AND protoPayload.methodName=*generativeai*" \
  --limit 1000 \
  --format=json \
  --project=hmatologia2 \
  | jq '.[] | {timestamp: .timestamp, tokens_in: .protoPayload.request.contents[0].parts[0].text | length, tokens_out: .protoPayload.response.candidates[0].content.parts[0].text | length}' \
  | awk '{in_cost += $NF * 0.000000075; out_cost += $(NF-1) * 0.00000030} END {print "Est. cost: $" in_cost + out_cost}'
```

### Script D: Weekly Metrics JSON Export

```bash
#!/bin/bash
# File: scripts/export-v1.4-weekly-metrics.sh
# Usage: bash scripts/export-v1.4-weekly-metrics.sh

WEEK=$(date +%Y-W%V)
OUTPUT_DIR=".planning/phases/$(printf '%02d' $(echo $WEEK | awk -F- '{print $NF}'))-weekly-metrics"

mkdir -p "$OUTPUT_DIR"

# Export Firebase metrics
firebase functions:log --region southamerica-east1 \
  --limit 5000 \
  --project hmatologia2 \
  > "$OUTPUT_DIR/firebase-functions-${WEEK}.log"

# Export GCP Billing snapshot
gcloud billing accounts list \
  --format=json \
  > "$OUTPUT_DIR/gcp-billing-${WEEK}.json"

# Export Cloud Logging errors
gcloud logging read \
  "severity=ERROR AND resource.type=cloud_function" \
  --limit 1000 \
  --format=json \
  --project=hmatologia2 \
  > "$OUTPUT_DIR/cloud-errors-${WEEK}.json"

echo "Metrics exported: $OUTPUT_DIR/"
echo "Files created:"
ls -lh "$OUTPUT_DIR/"
```
