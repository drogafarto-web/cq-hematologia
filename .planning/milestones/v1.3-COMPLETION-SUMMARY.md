## Lessons Learned (for v1.4)

1. **Functions API migration debt** — v1→v2 should have been a Phase 0 dependency. Caused all 121 TSC errors. v1.4 should run a `firebase-functions@latest` gate before any phase starts.
2. **Phase parallelism** worked at the *module* level, but cross-cutting concerns (shared schema, auth helpers, signature utils) need a designated owner — bias toward duplicated `shared/*` paths surfaced 24 module-resolution errors.
3. **Deferral honesty** — better to ship 70% well-tested than 100% with skipped E2E. Phase 10 + 11 deferrals are the right call.
4. **Riopomba migration** validated SGD multi-tenant approach. Use as template for next lab onboarding (Mercês, Tabuleiro).
