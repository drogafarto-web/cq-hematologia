### Task 4: ADR 0008 (0.5 dia)
- Criar `docs/adr/0008-bioquimica-westgard-clsi.md`:
  ```markdown
  # ADR 0008: Bioquímica usa subset CLSI clássico de Westgard (4 regras)

  **Status:** Accepted
  **Date:** 2026-XX-XX
  **Deciders:** CTO

  ## Context

  Módulo Hematologia usa 8 regras Westgard (1-2s, 1-3s, 2-2s, R-4s, 4-1s, 10x, 6T, 6X).
  Bioquímica precisa de Westgard mas com características diferentes:
  - Maior CV intrínseco em alguns analitos (GGT, TG)
  - Volume de runs por dia muito maior (centenas vs dezenas)
  - Padrão CLSI EP15 recomenda subset

  ## Decision

  Implementar **4 regras CLSI clássicas**: 1-2s (warn), 1-3s (reject), 2-2s (reject), R-4s (reject).
  Regras avançadas (4-1s, 10x, 6T, 6X) ficam definidas no código mas com flag `enabled: false`.
  UI admin em v1.4 permitirá ativar por analito.

  ## Consequences

  + Menos false-rejects em analitos com CV alto
  + Padrão internacional (CLSI EP15) seguido
  + Implementação mais enxuta
  + UX menos alarmista (operador não recebe rejeição em casos limítrofes)
  - Cobertura analítica menor que hema (intencional; trade-off aceito)
  - Pode precisar adicionar regras se auditor exigir (mitigado: framework permite extensão)

  ## Alternatives Considered

  - Set completo de hema (rejected: false-rejects em CV alto)
  - Configurável por analito desde v1.3 (rejected: complexidade UI desnecessária no MVP)

  ## References

  - CLSI EP15-A3
  - HC_Quality_Modulo_Hematologia_2026-04-29.md (Obsidian)
  - .planning/phases/09-bioquimica/CONTEXT.md
  ```

**Acceptance:** ADR aprovado pelo CTO; merge no main.
