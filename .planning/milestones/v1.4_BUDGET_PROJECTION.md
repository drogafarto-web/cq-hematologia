#### Dashboard Query Template (BigQuery)

```sql
-- Daily cloud spend by service (v1.4)
SELECT
  DATE(usage_start_time) AS date,
  service.description,
  SUM(cost) AS daily_cost,
  SUM(usage.amount) AS usage_volume
FROM `hc_quality_billing.gcp_billing_export_v1`
WHERE
  DATE(usage_start_time) >= '2026-05-20'
  AND DATE(usage_start_time) <= '2026-09-30'
  AND project.id = 'hmatologia2'
GROUP BY date, service.description
ORDER BY date DESC, daily_cost DESC;
```
