## How to Start Monitoring

### Option A: Auto-Start on System Boot (Recommended)

**Windows:**

Create a scheduled task:
```powershell
# In PowerShell (as Administrator)
$action = New-ScheduledTaskAction -Execute "pwsh" -Argument "-NoProfile -File '$((Get-Item -Path .).FullName)\scripts\v1.4-cascade-orchestrator.ps1'"
$trigger = New-ScheduledTaskTrigger -AtStartup
$principal = New-ScheduledTaskPrincipal -UserId $env:USERNAME -LogonType Interactive
Register-ScheduledTask -Action $action -Trigger $trigger -Principal $principal -TaskName "v1.4-Cascade-Monitor" -Description "v1.4 Cascade Orchestrator — Auto-monitor and trigger phases"
```

**macOS/Linux:**

Add to crontab:
```bash
# Run at startup
@reboot bash ~/work/hc-quality/scripts/v1.4-cascade-orchestrator.sh >> ~/cascade.log 2>&1 &
```
