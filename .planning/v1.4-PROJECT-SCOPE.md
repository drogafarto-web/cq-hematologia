# v1.4 Project Scope & Strategic Goals

**Bridge from internal compliance (v1.3) to external audit readiness (2026-08-31)**

---

## Mission

Deliver **audit-ready SaaS platform** for HC Quality with:
- **DICQ 88%+ conformance** (from 78.5%)
- **RDC 978/2025 100% critical articles** covered
- **LGPD full compliance** (consent + deletion + PII handling)
- **Auditor pre-alignment** (weekly calls + artifact approval)
- **Riopomba production stability** (80 docs, 1 lab, zero issues)
- **Portal ecosystem** (RT medical + patient external)
- **CAPA closure ceremony** (7 findings → auditor sign-off)

Target: Ready for **external ANVISA audit 2026-08-31**.

## Key Deliverables

### Portal Ecosystem (Phases 4–7)

**RT Medical Portal** (Phase 4):
- Login callable + JWT session management
- Laudo list (paginated) + draft access
- Review UI (dark-first, WCAG AA)
- Signature flow (LogicalSignature + chainHash)
- Audit trail (event versioning)
- Release candidate: state machine (Pendente → Em Revisão → Liberado)

**Portal Médico Completion** (Phase 6):
- PDF generation callable (puppeteer-based, CloudFunction 1GB/300s)
- QR validation + verification link
- SSO integration (optional Phase 6+)
- E2E coverage (8 flows)

**Patient External Portal** (Phase 7):
- Feedback submission form (public, no auth)
- NPS survey (post-resolution)
- Portal link access (email + QR)
- Trending dashboard (4.15 integration)

### Critical Value Escalation (Phase 5)

**Real-Time Alert System:**
- Callable: `setCriticalThreshold` (config)
- Callable: `escalateCriticalValue` (trigger)
- SMS/email integration (Twilio + SendGrid)
- SLA tracking (target <2 min delivery)
- Dashboard: escalation history + trending

### IA Training Dataset (Phase 5)

**Immunology Strip Annotation:**
- Upload UI (image batch)
- Gemini Vision integration (annotation)
- Dataset versioning (v1.0 baseline, v1.1+ experiments)
- A/B testing framework (model evaluation)
- Audit trail (dataset lineage)

### CAPA Closure Process (Phase 8)

**7 Audit Findings Resolution:**
1. F-01: Turnos auditável (Phase 0 impl → verify)
2. F-02: DPIA completo (Phase 0 impl → verify)
3. F-03: Lab-apoio contratos (Phase 0 impl → verify)
4. F-04: Risks FMEA formal (Phase 0 impl → verify)
5. F-05: Management review rotinizado (CTO + Auditor)
6. F-06: Documento distribuição aprovado (CTO + Auditor)
7. F-07: Rastreabilidade incompleta (CTO + Auditor ceremony)

**Auditor Ceremony:** 2026-08-05 sign-off + external audit ready confirmation
