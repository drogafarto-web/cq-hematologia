## Architecture

### Multi-Tenant Safety

- Job path enforces labId partition
- Queries filter by labId + soft-delete (deletadoEm === null)
- Cross-lab export impossible by design

### Async Model

- Client initiates → immediate jobId response → polls for completion
- Worker processes jobs independently via Pub/Sub
- Resilient to failures via exponential backoff retry

### Security & Compliance

- operatorId captured from auth token (RDC 978 § 5.3)
- Signed URLs: 7-day expiry enforces timely download
- All operations logged to Cloud Logging
- Multi-tenant isolation by Firestore rules
