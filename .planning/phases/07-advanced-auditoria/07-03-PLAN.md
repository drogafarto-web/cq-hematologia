**Pesos das 5 dimensões:**

| Dimensão | Peso | Cálculo |
|----------|------|---------|
| operation_rarity | 0.25 | `1 - counts[op] / max(total, 1)` |
| time_anomaly | 0.20 | `1 - hourly[hour] / max(total/24, 1)` |
| result_rarity | 0.20 | `falha→1.0, aviso→0.6, sucesso→0.0` |
| velocity | 0.20 | `0.0` (v1 — histórico requer janela) |
| module_jump | 0.15 | `0.0` (v1) |

`overall = Σ(score_d × weight_d)`, clamp [0.0, 1.0]. Se `overall > 0.85` → chamar `matchPatternWithAI`.

**Testes puros em `scoreEntry` (8 casos):** operação comum → score baixo, operação inexistente → score alto, resultado falha → result_rarity 1.0, hora 3am vs baseline diurno → time_anomaly alto, overall sempre [0,1], pesos somam corretamente, tudo normal → score baixo, tudo anormal → score próximo 1.0.

**Done:** exports + `npm test -- anomalyDetector` 100% + `npx tsc --noEmit` limpo.
