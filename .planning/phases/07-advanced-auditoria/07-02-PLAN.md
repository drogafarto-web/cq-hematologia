## SA-08 — `firestore.rules` (adicionar 2 blocos)

Leia o arquivo atual. Adicionar após o último match block existente. Preservar tudo.

```
// RDC 978 5.3 — audit trail imutável, leitura só por auditores
match /labs/{labId}/audit-trail/{entryId} {
  allow read: if isActiveMemberOfLab(labId) && getMemberRole(labId) in ['auditor','rt','admin','owner'];
  allow create: if request.auth == null && request.resource.data.labId == labId
                && request.resource.data.assinatura is map
                && request.resource.data.assinatura.hash.size() == 64;
  allow update: if false;
  allow delete: if false;
}
// DICQ 4.4 — alertas: leitura por auditores, dismiss por auditores
match /labs/{labId}/audit-alerts/{alertId} {
  allow read: if isActiveMemberOfLab(labId) && getMemberRole(labId) in ['auditor','admin','owner'];
  allow create: if request.auth == null && request.resource.data.labId == labId;
  allow update: if isActiveMemberOfLab(labId) && getMemberRole(labId) in ['auditor','admin','owner']
                && request.resource.data.diff(resource.data).affectedKeys()
                   .hasOnly(['status','dismissedAt','dismissedBy','dismissReason']);
  allow delete: if false;
}
```
