## SA-16 — `src/features/qualidade/components/AlertCenter.tsx`

**Modelo:** Haiku  
**Dependências:** `useAnomalyAlerts` (SA-12), `anomalyTypes.ts` (SA-01)  
**Contrato:** Dashboard de alertas dark-first. Lista de alertas ativos com badge de severidade, ordenados por score.

**Props:**

```typescript
interface AlertCenterProps {
  labId: string;
}
```

**Layout (dark-first, Tailwind):**

```
┌─ Alert Center ──────────────────────── [3 active] ─┐
│ 🔴 CRITICAL  Operador X — analyzer    score 0.97    │
│   Night-time result release outside shift hours     │
│   [Ver detalhes]                          [Fechar ▼]│
├─────────────────────────────────────────────────────┤
│ 🟡 HIGH      Operador Y — sgq         score 0.87    │
│   Rapid succession of document approvals            │
│   [Ver detalhes]                          [Fechar ▼]│
└─────────────────────────────────────────────────────┘
```

**Regras de implementação:**

- Usar `useAnomalyAlerts(labId)` para dados
- Badge de severidade: `critical` → vermelho (`bg-red-500/20 text-red-400`), `high` → âmbar, `medium` → amarelo
- `[Ver detalhes]` → abre `AlertDrillDown` (via state, não navegação)
- `[Fechar]` → expande input de motivo → chama `dismissAlert(alertId, reason)`
- Estado empty: `"Nenhum alerta ativo"` com ícone de check verde
- Loading state: skeleton de 3 linhas
- ≤200 LOC (sem comentários desnecessários)
- WCAG AA: contraste ≥4.5:1, focus ring visível, `aria-label` nos botões de ação

**Done:**

- [ ] `AlertCenter.tsx` existe, importa `useAnomalyAlerts`
- [ ] Renderiza lista de `AuditAlert[]` com badge de severidade
- [ ] Dismissal com input de motivo funcional
- [ ] Estado empty e loading presentes

**Modelo:** Haiku  
**Dependências:** `anomalyTypes.ts` (SA-01)

```typescript
interface AlertDrillDownProps {

**Layout (modal dark):**

```
┌─── Alerta: Operador X — analyzer ────────── [×] ──┐
│ Score: 0.97 CRITICAL                               │
│                                                     │

│                                                     │
│ ▸ Ação recomendada                                 │

- `onClose` chamado após dismiss bem-sucedido
- Sem navegação — é um modal (overlay com `fixed inset-0`)
- ≤180 LOC
