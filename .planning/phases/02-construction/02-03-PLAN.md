# Phase 2 Batch 3: Analítico + Pós-Analítico

**Scope:** 5 modules, >80% test coverage, gates after Batch 1  
**Inputs:** ADR 0004 (POPs), ADR 0007 (Equipamento), ADR 0003 (NC), Batch 1 (POPs + NC live)  
**Outputs:** 5 modules live, CIQ bioquímica + proficiency + method validation + report approval + critical comms

---

## Tasks

### Task 1: CIQ Bioquímica Module

**Duration:** 3-4 weeks  
New CIQ module (same pattern as Hematologia).

**Components:**

- CIQBioService: criarRunBio, salvarResultadoBio
- RunList: create, reagent tracking, results entry
- Validation: equipId (gate), popId (denormalization), operadorId (audit)

**Tests:** >80% (gates, denormalization)

**Commit:** "Batch 3 Task 1: CIQ Bioquímica live"

---

### Task 2: CEQ Module

**Duration:** 2-3 weeks  
External proficiency program with Z-score + auto-NC.

**Components:**

- CEQService: criarCEQ, receberAmostra, calcularZscore
- SampleTracking: enrollment, result entry
- AutoNC: if |Z| > 3, open NC (grave, blocking)

**Tests:** >80% (Z-score logic, auto-NC creation)

**Commit:** "Batch 3 Task 2: CEQ live"

---

### Task 3: Validação de Métodos

**Duration:** 2-3 weeks  
Method validation protocols (linearity, precision, accuracy, range).

**Components:**

- ValidacaoService: criarValidacao, registrarResultado, calcularAccuracy
- LinearityChart: r² visualization
- AccuracyReport: %recovery, %CV

**Tests:** >80% (validation formulas)

**Commit:** "Batch 3 Task 3: Validação de Métodos live"

---

### Task 4: Liberação de Laudos

**Duration:** 3-4 weeks  
Dual-signature approval + 5-year retention.

**Components:**

- LaudoService: criarRevisaoLaudo, aprovarLaudo (dual signature)
- CriticalResultList: criteria definition
- ApprovalUI: two operator review + sign

**Tests:** >80% (signature validation, retention policy)

**Commit:** "Batch 3 Task 4: Liberação de Laudos live"

---

### Task 5: Comunicação Resultados Críticos

**Duration:** 2 weeks  
Critical result criteria + communication registry + tracking.

**Components:**

- CriticosService: definirCriterioCritico, registrarComunicacao
- CriticalRegistry: auto-flag on criteria match
- NotificationUI: recipient tracking

**Tests:** >80% (criteria matching, notification logic)

**Commit:** "Batch 3 Task 5: Comunicação Críticos live"

### Task 6: Integration + Deployment

**Duration:** 1-2 weeks  
Cross-module smoke tests, rules deployment, prod deployment.
