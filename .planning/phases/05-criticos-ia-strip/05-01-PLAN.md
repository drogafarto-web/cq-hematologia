## Requirements

### REQ-01.1: Threshold Data Model & Firestore Binding

**Current schema** (Phase 3):

```
/labs/{labId}/criticos-thresholds/{id}
  ├─ analitoId: string (e.g., 'rbc', 'wbc')
  ├─ analitoNome: string (display name)
  ├─ min: number | null
  ├─ max: number | null
  ├─ unidade: string (e.g., 'M/uL')
  ├─ severidade: 'alta' | 'baixa'
  ├─ condicional?: { idadeMin?, idadeMax?, sexo? }
  ├─ ativo: boolean
  ├─ criadoEm: Timestamp
  ├─ criador: { operadorId, nome }
  └─ deletadoEm?: Timestamp (soft-delete)
```

**Validation rules:**

- `analitoId` must exist in analyte registry
- `min` + `max` cannot both be null
- `min < max` (if both present)
- `unidade` must match analyte default unit
- `condicional` fields (if present) must be logically consistent
- At least 1 threshold per analyte per lab

**Acceptance criteria:**

- [ ] Schema commented in Firestore
- [ ] Firestore Rules allow RBAC (RT + admin only)
- [ ] Soft-delete enforced (no hard delete)
- [ ] Indexes created (query by `analitoId`, by `ativo`)

**Zod schema** (src/features/criticos/types):

```typescript
const CriticoThresholdInput = z
  .object({
    analitoId: z.string().min(1),
    analitoNome: z.string().min(1),
    min: z.number().nullable(),
    max: z.number().nullable(),
    unidade: z.string().min(1),
    severidade: z.enum(['alta', 'baixa']),
    condicional: z
      .object({
        idadeMin: z.number().int().positive().optional(),
        idadeMax: z.number().int().positive().optional(),
        sexo: z.enum(['M', 'F']).optional(),
      })
      .optional(),
    ativo: z.boolean(),
  })
  .refine((data) => data.min !== null || data.max !== null, 'At least min or max must be specified')
  .refine((data) => !data.min || !data.max || data.min < data.max, 'min must be less than max');
```

### REQ-01.5: Firestore Rules + Indexes

**Rules requirement:**

```
/labs/{labId}/criticos-thresholds/{id}:
  - read: (isActiveMemberOfLab && lab.role in ['RT', 'admin']) || isSuperAdmin()
  - write: (isActiveMemberOfLab && lab.role in ['RT', 'admin']) || isSuperAdmin()
  - delete: never (soft-delete only)

/labs/{labId}/criticos-routing/{id}:
  - read: (isActiveMemberOfLab && lab.role in ['admin']) || isSuperAdmin()
  - write: (isActiveMemberOfLab && lab.role in ['admin']) || isSuperAdmin()
```

**Indexes needed:**

- `/labs/{labId}/criticos-thresholds` by `analitoId` + `ativo`
- `/labs/{labId}/criticos-routing` by `ativo`

**Acceptance criteria:**

- [ ] Rules deployed to staging + verified in emulator
- [ ] Indexes exist in Firestore (metadata)
- [ ] 0 rule violations in test suite
