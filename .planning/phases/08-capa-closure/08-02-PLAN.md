## SA-06 — `firestore.rules`

Adicionar blocos de regras para capa-tracking, calibracao, personnel. Edição — não recriar o arquivo.

**Blocos a adicionar (append após regras existentes):**

```
// capa-tracking: callable-only create/update; read = lab member; delete = never
match /labs/{labId}/capa-tracking/{capaId} {
  allow create: if false;  // callable only
  allow read: if isActiveMemberOfLab(labId);
  allow update: if false;  // callable only
  allow delete: if false;
  match /audit-log/{logId} { allow read: if isActiveMemberOfLab(labId); allow write, delete: if false; }
  match /evidence/{evidenceId} { allow read: if isActiveMemberOfLab(labId); allow write, delete: if false; }
}

// calibracao: callable-only write; read = lab member
match /labs/{labId}/calibracao/{recordId} {
  allow create, update: if false;  // callable only
  allow read: if isActiveMemberOfLab(labId);
  allow delete: if false;
}

// personnel/cargos: admin create/update; read = lab member
match /labs/{labId}/personnel/cargos/{cargoId} {
  allow create, update: if isAdminOrOwner(labId);
  allow read: if isActiveMemberOfLab(labId);
  allow delete: if false;
}

// personnel/designacoes: callable-only create; read = lab member
match /labs/{labId}/personnel/designacoes/{designacaoId} {
  allow create: if false;  // callable only
  allow read: if isActiveMemberOfLab(labId);
  allow update, delete: if false;
}

// management-review: callable-only write; read = admin/RT/auditor
match /labs/{labId}/management-review/{meetingId} {
  allow create, update: if false;  // callable only
  allow read: if isActiveMemberOfLab(labId) && (isAdminOrOwner(labId) || request.auth.token.role == 'RT');
  allow delete: if false;
}
```

**Invariantes:**

- Não remover nenhuma regra existente
- `isActiveMemberOfLab` + `isAdminOrOwner` já existem no arquivo

**Verify:** `firebase emulators:exec --only firestore 'npm run test:rules' 2>&1 | grep -E "(PASS|ok)"` ou `npx tsc --noEmit`
