## SA-37 — `src/features/risk-management/components/RiskMatrixHeatmap.tsx`

Heatmap visual de riscos NPR. ~110 LOC. **Novo componente** no módulo `risks` existente.

**Props:** `{ labId: string }`  
**Usa:** `riskMatrixService.getRisks(labId)` via `useEffect` (getDocs one-time)

**UI:**
- Grid 5×5: eixo X = Probabilidade (1–5), eixo Y = Severidade (1–5)
- Cor por NPR médio da célula: <25 emerald, 25–50 amber, >50 red
- Cada célula: conta riscos naquela P×S + hover mostra nomes
- Lista abaixo: top 5 riscos por NPR (nome + NPR + controles resumidos)
- Botão "Ver todos os riscos" → link para página de risks existente

**Invariantes:**
- NPR = P × S × D (D fixo 3 se não informado)
- Sem lib de charts — CSS grid puro
