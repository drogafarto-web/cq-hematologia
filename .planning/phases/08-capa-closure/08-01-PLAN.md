## SA-01 — `src/features/capa-tracking/types/index.ts`

Type system unificado para CAPA closure. Zero lógica — apenas interfaces, enums e helpers puros.

**Exports obrigatórios:**

```typescript
export type CapaState = 'open' | 'in-progress' | 'evidence-submitted' | 'auditor-reviewing' | 'closed';
export type CapaSeverity = 'critical' | 'major' | 'minor';
export type RFIStatus = 'pending' | 'answered';

export interface LogicalSignature { hash: string; operatorId: string; ts: number; }

export interface Evidence {
  id: string; labId: string; capaId: string;
  fileName: string; fileSize: number;
  mimeType: 'application/pdf' | 'image/png' | 'image/jpeg' | 'text/plain';
  storagePath: string; uploadedBy: string; uploadedAt: number;
  hash: string; signature: LogicalSignature;
  description?: string; deletedAt?: number;
}

export interface AuditorRFI {
  rfiId: string; question: string; askedBy: string; askedAt: number;
  dueDate: number; response?: string; respondedBy?: string;
  respondedAt?: number; status: RFIStatus; deletedAt?: number;
}

export interface CapaFinding {
  findingId: string; title: string; severity: CapaSeverity;
  dicqBlocks: string[]; rdcArticles: string[];
}

export interface CapaStateTransition {
  from: CapaState | null; to: CapaState;
  transitionedAt: number; transitionedBy: string; reason?: string;
}

export interface CapaDocument {
  id: string; labId: string; finding: CapaFinding; state: CapaState;
  createdAt: number; createdBy: string;
  rootCause: string; correctiveAction: string;
  deadlineDate: number;
  evidence: Evidence[]; rfiLog: AuditorRFI[]; stateHistory: CapaStateTransition[];
  auditorSignOffEmail?: string; auditorSignOffAt?: number;
  deletedAt?: number;
}

// Helpers puros
export function isValidStateTransition(from: CapaState, to: CapaState): boolean;
export function daysRemaining(deadlineDate: number): number;
```

**Invariantes:**
- `isValidStateTransition`: `closed` não tem transições válidas; `auditor-reviewing` pode voltar para `in-progress`
- Sem imports externos além de Zod (opcional para schemas)
- 0 lógica de negócio — apenas tipos + helpers puros

**Verify:** `npx tsc --noEmit 2>&1 | grep -c error | grep -E '^0$'`
