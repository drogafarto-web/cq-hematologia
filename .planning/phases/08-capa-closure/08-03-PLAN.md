```

```

**Step 2: Auditor Reviews Evidence (3–5 business days)**
- Auditor logs into portal
- Views CAPA tracking dashboard (6 closed + 6 deferred visible)
- For each closed CAPA:
  - Reviews root cause analysis (markdown)
  - Reviews evidence artifacts (PDFs, certificates, screenshots)
  - Reviews audit logs (all actions timestamped + operator-tracked)
  - Checks CAPA state transitions (immutable history)
- For each deferred CAPA:
  - Reads phase plan
  - Reviews mock evidence (proof of concept)
  - Checks timeline (phase target dates)

**Step 3: RFI Submission (if needed)**
If auditor has questions:
- Click "Submit RFI" button
- Form appears with pre-populated CAPA id
- Auditor types question + due date
- System emails question to CTO + QA Manager
- Response deadline = due date (target: within due date)

**Step 4: RFI Resolution (iterative, <3 cycles target)**
- Lab receives RFI
- QA Manager/CTO respond via form (or email forwarded to callable)
- Response logged in rfiLog (immutable)
- Auditor reviews response + confirms satisfied (or submits follow-up RFI)

**Step 5: Auditor Sign-Off (final)**
Once auditor satisfied:
- Auditor opens CAPA dashboard
- Clicks "Sign Off on Phase 8 CAPAs"
- Form appears:
  - Message: "I have reviewed the evidence for all 12 CAPAs and find them acceptable for closure."
  - Auditor name + firm + date
  - Signature (digital signature via FirebaseAuth or email-based)
- Submit button
- System logs:
  - Auditor email + timestamp
  - All 12 CAPAs state → 'closed' (final state)
  - Audit log entry: { action: 'auditor-signoff', timestamp, auditorEmail }
  - Creates Firestore doc: `/capa-tracking/auditor-signoff` with confirmation + PDF

- Dashboard: lists 6 closed CAPAs + 6 deferred
- Detail view: root cause + evidence artifacts (PDFs embedded or linked)
- RFI form: inline on detail page
- Sign-off form: appears after RFI log empty (or auditor confirms ready)

**Callable: submitAuditorSignOff**
- Input: { labId, auditorEmail, auditorName, auditorFirm, message, signature }

  1. Query all 12 CAPAs (6 closed + 6 deferred)

     ```
     Subject: CAPA Closure Sign-Off Confirmed

     Auditor: {auditorName}, {auditorFirm}
     Sign-off Date: {timestamp}
     CAPAs Closed: 12 (6 immediate + 6 deferred with phase plans)

     Next: Phase 9 execution begins. Deferred CAPA progress tracked via phase plans.
     ```

  5. Log to audit trail
