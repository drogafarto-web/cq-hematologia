## SA-14 — `src/features/capa-tracking/hooks/useCapaTracking.ts`

Hook para lista + seleção de CAPAs em tempo real.

**Exports:**

```typescript
export function useCapaTracking(
  labId: string,
  filterState?: CapaState | 'all',
): {
  capaList: CapaDocument[];
  loading: boolean;
  error: string | null;
  selectedCapaId: string | null;
  selectCapa: (id: string | null) => void;

```

**Invariantes:**

- `useEffect` retorna `unsubscribe()` chamado no cleanup

**Verify:** `npx tsc --noEmit 2>&1 | grep -c error | grep -E '^0$'`
