## SA-11 — `src/features/bioquimica/hooks/useAcceptanceEngine.ts`

Hook que integra `evaluateRules` do SA-05 para um analito específico em tempo real.

**Export:**
```typescript
export interface UseAcceptanceEngineReturn {
  points: number[];
  violations: RuleViolation[] | null;
  status: "ok" | "warn" | "reject";
  addPoint: (value: number) => void;
  clearPoints: () => void;
}

export function useAcceptanceEngine(
  analito: Analito | null,
  nivel: { expectedValue: number; nivelId: string } | null
): UseAcceptanceEngineReturn
```

**Comportamento:**
- `addPoint(value)`: acrescenta ao array de pontos, chama `evaluateRules(points, analito.westgardRules, sigma)`
- `sigma` computado de `analito.cvIntra` se disponível: `sigma = nivel.expectedValue * (cvIntra / 100)`; fallback: `1.0`
- `status`: "reject" se qualquer violação com severity reject, "warn" se só warn, "ok" se null
- `clearPoints()`: reseta tudo
- Se `analito === null`: retornar estado vazio (`points: [], violations: null, status: "ok"`)

**Min lines:** 80  
**Commit:** `feat(bioquimica): wire useAcceptanceEngine to Westgard evaluateRules (Phase 9 W2)`  
**Verify:** `npx tsc --noEmit src/features/bioquimica/hooks/useAcceptanceEngine.ts`
